# ML Portfolio

A collection of machine learning projects built in Jupyter notebooks.

## Projects
- Project 1: Boston House Price Prediction
- Project 2: Customer Churn Classification
- Project 3: Fashion Products Classification using Artificial Neural Networks (ANN)
- Project 4: Housing Price Prediction
- Project 5: Famously renowed IRIS Dataset
- Project 6: Image Classification between Dogs and Cats using Keras
- Project 7: Insurance Fraud Detection classification between good and bad customers
- Project 8: Classification of materials for their superconductivity by analysing their breaking temperatures.
- Project 9: Stock Market prediction of S&P 500 using Machine Learning
- Project 10: Apple Stock prediction using Stacked LSTM
- Project 11: Classification of good, bad and worst wine based on their quality.
